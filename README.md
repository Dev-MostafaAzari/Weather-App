# پروژه Weather App
## تکنولوژی استفاده شده:
- HTML
- Css
- JavaScript
- React
    - Redux
    - axios
- Framer Motion
## قابلیت های پروژه
- Responsive/Dynamic Design
- Show Weather Conditions
- Search Locations
- Animated background (ForEach Weather Condition)
