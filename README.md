# پروژه Weather App
## تکنولوژی استفاده شده:
- HTML
- Css
- JavaScript
- React
    - Redux
    - axios
- Framer Motion
## قابلیت های پروژه
- Responsive/Dynamic Design
- Show Weather Conditions
- Search Locations
- Animated background (ForEach Weather Condition)
## توضیح پروژه
<p align="right">
در این پروژه از ساختاری ساده ولی پویا در طراحی ظاهری استفاده شده است و این امکان را به کاربر می دهد تا با وارد کردن نام شهر یا منطقه مورد نظر اطلاعات آب و هوایی آن را دریافت و مشاهده کند همچنین چندین افکت آب و هوایی برای وضعیت آب و هوا در نظر گرفته شده است
</p>
